Displaying data

  1. Set up a firebase project. In database part (don't use messaging API for now) create a collection of messages and put some messages in there. Each message should have at least:
    • a sender
    • timestamp (very important for sorting!)
    • body
  2. If required, grant both read and write access to firebase db for everyone (in rules)
  3. Set up new Angular project
  4. Show properly sorted messages with all data (it's preferred to use firebase SDK for that)
